## Setup project

install dependencies

`npm install`

compile contracts

`npm run compile`

deploy contracts

`npm run deploy`

run tests

`npm run test`

run scripts

`npx hardhat run ./src/scripts/setAndGetName.ts`

run node in console

`npx hardhat node`
